# TechStore v3 — Cyber Violet Clean (демо)
Single-page demo: Catalog, About, Contacts (placeholders).

## Запуск
Откройте `index.html` в браузере.

## Особенности
- Тёмный футуристичный дизайн (Cyber Violet)
- Корзина закрывается при клике вне неё
- 6 товаров, картинки с picsum.photos
- Контакты — заглушки
- No loading spinner (страница открывается сразу)
